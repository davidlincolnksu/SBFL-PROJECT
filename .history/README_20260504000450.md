SBFL JavaFX Project

A Java desktop app that ranks methods by suspicion score using spectrum-based fault localization (SBFL). It reads test coverage data, computes scores using your chosen formula, and displays results ranked from most to least suspicious with color-coded rows.


REQUIREMENTS

    Java 17 or higher
    Maven 3.6 or higher

Verify with:
    java -version
    mvn -version


HOW TO RUN

    mvn javafx:run


APPLICATION USAGE

1. Click Load Data to load the CoverageData folder automatically.
   The status bar will show how many tests, methods, and failures were found.

2. Select a formula from the dropdown: Tarantula, Ochiai, Jaccard, or Zoltar.

3. Click Run SBFL to rank all methods by suspicion score.
   Results appear in the table ranked highest to lowest.
   Red rows are more suspicious. Green rows are less suspicious.

4. The overview cards at the top show a summary of the loaded dataset.
   Hover over a card for more detail.

5. The Log section at the bottom can be expanded to see a detailed
   record of each action including top suspect and score.
